# Dolf Combat

A comprehensive Dutch combat configuration for FiveM.

## Features

- **Adjustable Recoil**: Customize recoil settings for each weapon individually.
- **Adjustable Screen Shake**: Set unique screen shake levels per weapon.
- **Enhanced Grip Effects**: Reduced screen shake when a player has a grip attachment on their weapon.
- **Instant Headshot Death**: Headshots result in instant death across all weapons.
- **First-Person Only Mode**: Experimental forced first-person view (Beta).

- **All-in-One Resource**: Consolidated combat settings for easy management.

## Installation

1. Drag the `dolf_combat` folder into your server resources.
2. Add the following line to your `server.cfg` file:
   ```cfg
   ensure dolf_combat
   ```
